# teruo.dev

# dependencies

https://jekyllrb.com/docs/installation/

```bash
ruby -v
gem -v
gcc -v
g++ -v
make -v

gem install jekyll bundler
bundle config set --local path 'vendor/bundle'
bundle install
```

# up and running

```bash
bundle exec jekyll serve --livereload
```

access: http://localhost:4000
